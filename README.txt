/**************************/
/****      HOW TO      ****/
/**************************/

0 - The document "solid_propellant_combustion_model.docx" found in the "docs" folder contains the description of this model. 

1 - How to run the simulation examples for various solid propellant grain geometries
	
	1.1 - Prior to run any scripts, ensure that SIMU.exe is available in the "scripts" folder.
	1.2 - Each script (.bat file) found in the "scripts" folder run a specific model based on the desired pixel resolution (i.e., 20 x 20, 100 x 100, 200 x 200, 300 x 300, 400 x 400, or 500 x 500), and if using hexagonal rules or square rules.
	1.3 - All models are using "initialCellsValue" equals to: "../inputs/solid_propellant_input.val". In each script, the correct .val file representing a specific grain geometry is copied into the solid_propellant_input.val.
	1.4 - IMPORTANT NOTE: Only one script can be run at a time since all scripts are copying the .val file into the same input file: ../inputs/solid_propellant_input.val
	1.5 - Note that running any simulations using a pixel resolution above 200 requires significant computation time.
	1.6 - Once the simulation ends, a .log file will be stored in the "results" folder.
	
2 - How to visualize the results using the ARSLab DEVS Web Viewer

	2.1 - Go to https://staubibr.github.io/arslab-prd/app-simple/index.html
	2.2 - Click on the icon with the two arrows pointing in opposite directions.
	2.3 - Select CD++ and Cell-DEVS.
	2.4 - Drag and drop the following files pertinent to the experiment, then click "Parse files".
			1. .val file (found in the "inputs" folder)
			2. .pal file (found in the "color" folder)
			3. .log file (found in the "results" folder)
			4. .ma file (found in the "models" folder)
